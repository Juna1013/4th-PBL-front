# 機能詳細 (Features)

## 音声認識ダッシュボード

Raspberry Pi Pico W を制御するための音声コマンドを認識・表示するメインインターフェースです。

### 1. サーバー接続状態監視
- バックエンドサーバー (`http://localhost:8000`) との接続状態を定期的に確認します。
- **接続済み (緑)**: 正常に通信できています。
- **未接続 (赤)**: サーバーがダウンしているか、ネットワークエラーが発生しています。

### 2. 音声入力 (マイク)
- ブラウザのマイクアクセス機能を使用し、リアルタイムで音声をキャプチャします。
- **オーディオビジュアライザー**: 入力音量をリアルタイムでグラフ表示し、マイクが正しく音を拾っているか視覚的に確認できます。

### 3. 音声ファイルアップロード
- 録音済みの音声ファイルをアップロードして認識させることができます。
- バックエンドの `/recognize` エンドポイントに送信され、解析結果が表示されます。

### 4. コマンドシミュレーション (テスト用)
- マイク入力なしで動作確認を行うためのボタン群です。
- 以下のコマンドを即座に発行・表示できます：
  - 前進 (Blue)
  - 後退 (Yellow)
  - 左折 (Red)
  - 右折 (Green)
  - 停止 (Gray)
  - スタート (Purple)

### 5. 認識結果表示
- 最新の認識されたコマンドを大きく表示します。
- **信頼度 (Confidence)**: 認識の確からしさをパーセンテージとバーで表示します。

### 6. 履歴 (History)
- 過去の認識結果をリスト表示します。
- コマンド名、タイムスタンプ、信頼度が記録されます。
